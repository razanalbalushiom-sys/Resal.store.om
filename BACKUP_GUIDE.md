# Resal Store Backup

Use the admin dashboard to download a JSON backup:

1. Open `/admin.html`.
2. Log in as admin.
3. Open `Settings`.
4. Click `Download Backup`.

The backup includes products, orders, settings, and safe user account fields.

For Supabase-level backups, also enable scheduled backups from the Supabase dashboard when the project moves to a paid plan.
